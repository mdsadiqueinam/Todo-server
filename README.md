## JWT Bearer Authentication System

Jwt authentication system use `public` and `private` key combination with `RS256`. `id_rsa_priv.pem` and `id_rsa_pub.pem` files both contains the private and public key for authentication

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
